# MTX Vibe Coding 任务复盘

我最大的收获是可以让多个 AI Agent 分工协作迭代项目。同样这也是一次小项目的经历，让我对开发更熟悉。

---

协作中最困难的是音频和网页外观设计。AI 的设计太拉了。我通过找到更好的 skill、自己查找真实网页做参考解决了这个问题。

---

AI 很大地改变了产品开发。一个是技术要求大大降低了，我不会写代码，也可以指挥 AI 从头编程、测试、优化代码，以及教我如何完整地开发。现在的 Agent 甚至可以链接外部平台直接部署，并且在“协作体验”这方面也有大大提升。人的作用一个是结合自己的经验和理解，纠正 AI 的项目创意和方向，并提供细节的反馈和建议。还有一点是美术设计，人仍需起到主要作用。
