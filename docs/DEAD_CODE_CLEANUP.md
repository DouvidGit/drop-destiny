# Dead Code Cleanup

最终收尾阶段对旧界面与兼容代码进行了逐项审查。已删除的主要内容包括：

- 已退役的 Pattern / Arrangement / Live Drop 界面与状态分支；
- 旧 DNA 雷达图、分数条、观众反应组件及相关样式；
- 不再使用的可视化文字、装饰飞条与场景元数据；
- 已无调用者的旧数据、渲染函数和事件绑定。

保留了仍被音频停止流程、开发测试或兼容输入使用的接口。早期自动审计曾把 `playPattern()` 误记为最终歌曲的内部调用，因此旧报告没有直接作为最终文档使用；所有删除项均以实际引用搜索和回归测试结果为准。

清理后运行风格回归、音频引擎、结局集成、桌面布局和可视化测试，最终结果见仓库当前版本与提交包验收输出。
