## Bass wavetables

The single-cycle waveforms in `wavetables.js` are selected from
[Adventure Kid Wave Forms](https://github.com/KristofferKarlAxelEkstrand/AKWF-FREE),
released under [CC0 1.0](https://creativecommons.org/publicdomain/zero/1.0/).

Selected source waveforms:

- `AKWF_distorted_0025`
- `AKWF_fmsynth_0026`
- `AKWF_granular_0008`
- `AKWF_bitreduced_0002`
- `AKWF_hvoice_0050`

## Display font

The interface embeds **Teko** as a local WOFF2 file at
`assets/fonts/Teko.woff2`, so the project does not request an online font at
runtime. Teko is distributed under the SIL Open Font License 1.1. The complete
license text is included at `assets/fonts/OFL.txt`.
